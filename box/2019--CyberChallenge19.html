Networking fundamentals
 – How a network device communicates over TCP/IP. (first 4 layers of TCP stack) – How traffic flows between networks/subnets. – Network commands and tools – The “route” and “ip” commands. Some “netstat”. – Nmap, including outside of local subnet.

--------------------------------------------------------------------------------------------------------------------------------------
Ping Sweep:
nmap -PS [IP RANGE]

Service Scan:
nmap -sV [IP ADDR]

Service Scan for UDP:
nmap -sU [IP ADDR]

OS detection:
nmap -v -O --osscan-guess [IP ADDR]

Host protected by a firewall:

nmap -PN [IP ADDR]

fast scan:
nmap -T5 [IP RANGE]

Personal Fav:

nmap -A -p- [IP ADDR]

IP ROUTE:
ip route add [target network] via [intermediate router]

SCAN FOR SMB VULN:

nmap --script smb-vuln* [IP ADDRESS]
